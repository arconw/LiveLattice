# k6 Baseline

Baseline capture date: 2026-06-24

## Smoke Profile

Run command:

```bash
K6_PROFILE=smoke bash k6/run-all.sh
```

The smoke profile is the required local verification gate for Stage 14. It verifies public `/health` and `/ready` paths across gateway, core, realtime, search, notifications, import-export, audit-log, and background-jobs, plus auth-boundary failure paths. When `K6_AUTH_USERNAME` and `K6_AUTH_PASSWORD` or `K6_AUTH_TOKEN` are supplied, it also exercises authenticated core, API-key, dashboard, import-export, search, notifications, audit-log, and background-jobs flows.

## Baseline Profile

Run command:

```bash
K6_PROFILE=baseline bash k6/run-all.sh
```

The baseline profile keeps the prompt targets configurable through environment variables:

| Script | Default baseline profile |
|---|---|
| `rest-api.js` | 1000 requests per second for 5 minutes. |
| `canvas-collaboration.js` | 50 VUs for 3 minutes against realtime reachability. |
| `dashboard-queries.js` | 100 VUs for 3 minutes. |
| `import-export.js` | 10 VUs for 3 minutes. |
| `websocket-reconnect.js` | 100 VUs for 3 minutes. |
| `kafka-lag.js` | 5000 iterations per second for 1 minute with Prometheus lag sampling. |

Do not run the baseline profile on a shared local machine without confirming capacity.
